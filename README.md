# intellisql-intelligent-sql-querying-with-llms-using-gemini-pro
IntelliSQL is a Streamlit‑powered platform that leverages Google Gemini Pro to transform natural language questions into precise SQL queries. By combining LLM capabilities with a SQLite backend, it enables users to interact with databases intuitively, making data access faster, smarter, and more accessible.
